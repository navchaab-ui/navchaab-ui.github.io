Миний тухай

Би вэб хөгжүүлэгч бөгөөд орчин үеийн, хэрэглэгчид ээлтэй систем бүтээх сонирхолтой. PHP, HTML, CSS, React ашиглан веб аппликейшн хөгжүүлдэг. Одоогоор ажил олгогч, ажил хайгчдыг холбох систем болон ухаалаг зогсоолын систем дээр ажиллаж байна.
